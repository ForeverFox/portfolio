Portfolio Project

This was the final project for my Responsive Design course with the IDM Program at MCAD. The emphasis of this project was on designing and developing a mobile-first, responsively designed site. I opted to use CSS Grid for the layout, and developed a fullscreen horizontal split-screen design where the user could navigate using the left side of the screen to reveal content on the right side. To maximize responsiveness and improve user experience, the tablet/mobile view abandons the horizontal design in favor of a more traditional vertical design.

I hand-coded this site using HTML, CSS, and Sass.
