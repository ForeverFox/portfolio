Gili Estlin Hirsch

Website Redesign Project

This was the capstone project for my Web Design: Principles course with the IDM Program at MCAD. The purpose of this project was to showcase fundamental design principles such as typography, color theory, and grid-based design through the construction of a complete website. For my project, I chose to redesign an existing website that I had previously assembled (2013) for author Gili Estlin Hirsch.

I hand-coded this site using HTML, CSS, JavaScript and jQuery.
